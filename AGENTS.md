# Test 
  + Les tests exécutés avec la commande : `uv run pytest -v` à la racine du dépot 

# Documentation 
  + La documentation est construite avec la commande : `uv run make -C doc html` à la racine du dépot 

# Développement 
  + Tu utilises au maximum des objets OpenTURNS mais garde une compatiblité avec numpy 
  + Documentation OpenTURNS : https://openturns.github.io/openturns/latest/user_manual/user_manual.html
